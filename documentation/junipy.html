API Documentation 📝

This documentation covers the coffe shop order web API.

Media Type support

All server response bodies and request bodies MUST be valid JSON messages.

Home ☕️

The Home URL (/home/) is the base URL for reading and updating Order objects The table below shows all the Task-related operations, their URLs, methods, and payloads.

Operation URL Method Returns Inputs
fetch homepage resource /home GET GDP&Flags None

Detail 🍣

The Detail URL (/detail/<code>/) is the base URL for get country detail objects and supports the following operations:

Operation URL Method Returns Inputs
get the detail of the country /detail/<code>/ GET detail country code

Responses 👀

The home API returns payloads (as JSON object):

  • GDP&flags
  • Detail

See below for details.
