API Documentation 📝

This documentation covers the GDP anaylasis web API.

Media Type support

All server response bodies and request bodies MUST be valid JSON messages.

Home ☕️

The Home URL (/home/) is the base URL for reading GDP objects The table below shows all the Task-related operations, their URLs, methods, and payloads.

Operation URL Method Returns Inputs
fetch homepage resource /home GET GDP&Flags None

Detail 🍣

The Detail URL (/detail/<code>/) is the base URL for get country detail objects and supports the following operations:

Operation URL Method Returns Inputs
get the detail of the country /detail/<code>/ GET detail country code

Analysis 🍰

The Analysis URL (/analysis/) is the an URL for get analysis information for global trend and supports the following operations:

Operation URL Method Returns Inputs
get analysis data /analysis/ GET analysis None

Region 🍺

The Region URL (/regions/) is the URL for get analysis information for each region and supports the following operations:

Operation URL Method Returns Inputs
get the detail of the region /region/ GET analysis region code

Responses 👀

The home API returns payloads (as JSON object):

  • Home
  • Detail
  • Analysis
  • Region

See below for details.

Data Elements 🍣

There is all the parameters describtion

Home

Name Meaning Type Optional
GDP A list of latest year countries GDP List ✖️
flags Flags list of countries List ✖️

Detail

name country full name String ✖️
introduction country’s short introduction String ✖️
flag URL of large flag picture String ✖️

agriculture latest year agriculture String ✖️

agricultureHistory {year:,value:} String:String ✖️

industryHistory {year:,value:} String:String ✖️
service latest year service String ✖️
servicePrecent latest year service percentage String ✖️
serviceHistory {year:,value:} String:String ✖️
CO2 latest year CO2 emission String ✖️
CO2History {year:,value:} String:String ✖️
PM25Index latest year PM2.5 String ✖️
