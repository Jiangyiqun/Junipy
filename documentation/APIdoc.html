API Documentation 📝

This documentation covers the coffe shop order web API.

Media Type support

All server response bodies and request bodies MUST be valid JSON messages.

GDP

The GDP URL (/GDP) and (/SortedGDP)are the base URL for get the GDP of the latest year(2016).The table below shows all the Task-related operations, their URLs, methods, and payloads.

Table 1. GDP operations

Operation URL Method Returns Inputs
get latest year GDP /GDP GET country short code,raw GDP none
get latest year GDP(0-1) /SortedGDP GET country short code,0-1 GDP none

Counrty details

The country details URL (/detail/) is the base URL for get different countrys' details objects and supports the following operations:

Table 2. country detailoperations

Operation URL Method Returns Inputs
GetCountryDetail /detail/<code> GET countrydetail country code

Flags

The flag URL (/flags) is the base URL for get all countrys' flags objects and supports the following operations:

Table 2. country detailoperations

Operation URL Method Returns Inputs
GetAllCountrys' Flags /falgs GET countrys' names and flags none

 

Responses 👀

The GDP API returns 3 possible payloads (as JSON object):

  • Latest GDP
  • details
  • Flags

See below for details.
